## 🔍 主要内容

### 一、涨价通胀成为主线的逻辑

**三大原因**:

| 原因 | 说明 |
|------|------|
| **标杆效应** | AI产业链因供需紧张持续涨价，为市场树立样板 |
| **位置优势** | 多数周期品经历几年通缩，位置低、性价比高 |
| **资源事件** | 津巴布韦等矿产资源政策变化，催化资源品涨价预期 |

### 二、化工行业核心逻辑

**行业背景**:
- 多数化工品经历近4年下滑，产品价格和股价位置较低
- 从被动去库进入主动补库阶段，逐步迎来拐点

**投资主线**:

| 主线 | 关注方向 | 代表标的 |
|------|---------|---------|
| **戴维斯双击** | 景气赛道+卡脖子品种 | AI相关（电子化学品） |
| **资源类** | 矿产资源品 | 磷矿、钾肥、萤石 |
| **龙头白马** | 资本上台阶，弹性大 | 万华化学、华鲁恒升 |
| **边际变化** | 迎来拐点的品种 | 氨纶、有机硅 |
| **底部品种** | 利空出尽，极度承压 | PVC、钛白粉 |
| **双碳受益** | 能耗管控受益 | 黄磷等 |

### 三、重点品种分析

#### 1. MDI/TDI（万华化学）

| 指标 | 数据 |
|------|------|
| 2024年业绩 | 100亿（预测125亿） |
| 2025年预期 | 160亿（不涨价）→ 210亿（涨价1500元） |
| 产能 | MDI+TDI 400万吨 |
| 弹性 | 涨价1000元 → 增厚34亿利润 |
| 当前市值 | 2800亿（13倍PE） |
| 目标空间 | 30%-70% |

**逻辑**: 寡头垄断格局，海外持续提价，趋势票

#### 2. 甜味剂（金禾实业）

| 品种 | 格局 | 涨价预期 | 利润弹性 |
|------|------|---------|---------|
| 安赛蜜 | 金禾+醋化占80-90%产能 | 涨3万以上 | 4亿→7亿+ |
| 三氯蔗糖 | 金禾+科宏+康宝三家 | 参考2024年涨到20万 | 8亿利润增量 |

**目标**: 利润16亿，250亿市值（15-16倍PE）

#### 3. 有机硅

| 要点 | 内容 |
|------|------|
| 供需 | 2026年无新增产能，2027年不确定 |
| 海外 | 陶氏14.5万吨产能6月退出 |
| 价格 | 当前1.4万，企业有盈利 |
| 历史高点 | 3万以上（自身周期），6万（含工业硅影响） |
| 会议 | 2月28日行业会议定全年基调 |

**推荐标的**: 东岳硅材（纯度最好）、新安股份、合盛硅业、兴发集团

#### 4. 磷化工

| 环节 | 现状 | 预期 |
|------|------|------|
| **磷矿** | 供应春耕，情绪乐观 | 阶段性高品位矿小幅提价 |
| **磷肥** | 国内保供，出口受限 | 春耕后关注出口政策变化 |
| **黄磷** | 能耗高，供给稀缺 | 硫磺价格高支撑，农药出口预期提振 |

**催化**: 美国含磷农药供应紧张，可能带来出口机会

#### 5. 钛白粉 & PVC

| 品种 | 现状 | 催化 |
|------|------|------|
| **钛白粉** | 龙佰2月28日提价500元/吨（国内），100美元/吨（海外） | 金三银四旺季，涨价落实概率大 |
| **PVC** | 4月1日取消出口退税 | Q1抢出口，关注落后产能出清 |

**特征**: 均处于历史大周期底部，向下风险小，向上弹性大

#### 6. 氨纶 & PTA

| 品种 | 逻辑 |
|------|------|
| **氨纶** | 需求好转+潜在供给出清，多次推荐的涨价品种 |
| **PTA** | 产能集中，规模化+一体化优势，关注民营大炼化（荣盛、恒力） |

### 四、投资结论

1. **主线明确**: 涨价/通胀是核心主线，化工板块机会多
2. **方向多元**: 既有AI景气赛道，又有传统周期，还有资源品
3. **重点标的**:
   - 趋势票：万华化学（MDI/TDI）
   - 价值票：金禾实业（甜味剂）
   - 弹性票：有机硅、磷化工、钛白粉、PVC
   - 拐点票：氨纶
