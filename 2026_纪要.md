## 🔍 主要内容

### 一、春节消费数据解读

**数据表现**:
- 春节零售餐饮日均消费同比 +5.7%（2025年为+4.1%）
- 好于预期，但理性消费趋势未改
- 价格仍敏感，人均旅行消费预算未有效提升

**驱动因素**:
- 9天长假效应（比惯常多1天）
- 暖冬天气
- 底层消费意愿触底恢复

**核心判断**: 消费触底后恢复缓慢波动，通缩缓慢转向低速CPI增长

### 二、2026年四大投资主题

| 主题 | 板块 | 时间节奏 |
|------|------|---------|
| **线下服务恢复** | 旅行、酒店、餐饮、啤酒 | 餐饮啤酒拐点在Q2-Q3 |
| **价格恢复** | 餐饮啤酒(Q1)、体育用品化妆品(H2)、白酒(H2) | 分阶段 |
| **供给收缩** | 乳制品 | Q2后拐点 |
| **海外增长** | 内容出海、服饰OEM | 全年 |

### 三、分板块分析

#### 1. 餐饮 - 推荐海底捞

| 指标 | 数据/判断 |
|------|----------|
| 春节翻台率 | +5%（超预期） |
| 低基数 | 4月开始（去年4月外卖平台价格战） |
| EPS预期 | 2025-2027年CAGR 27% |
| 分红 | 95%分红率 |
| 投资逻辑 | 同店转好→开店模型恢复→PE扩张 |

#### 2. 黄金珠宝 - 老铺黄金

- 克重产品销售弱（金价波动大）
- 一口价产品销售好（价格未变）
- 2月28日提价，提前布局导致销售火爆
- **风险**: 提价后4-11天股价通常走弱（排队效应消失）

#### 3. 海南免税

| 指标 | 数据 |
|------|------|
| 春节销售增长 | +16.5% |
| 客单价 | -4%（人多但人均消费降） |
| 全年预期 | 20-30%增长 |

**判断**: 16%增长不差（基数高），旅游OK但旅游零售弱于旅游体验

#### 4. 酒店 - 推荐华住、亚朵、锦江、首旅

| 指标 | 数据 |
|------|------|
| RevPAR | 连续4个月转正，春节+3%~+5% |
| vs 2019年 | 恢复至110-112% |
| 驱动因素 | 价格带动（人数增长平缓） |
| 区域表现 | 华东华南好，华北差 |

**推荐逻辑**:
- 龙头（华住、亚朵）: 净利润增长20%+
- A股（锦江、首旅）: 从负增长转正10%，补涨空间大

#### 5. OTA - 携程

| 指标 | 数据 |
|------|------|
| Q4收入 | beat 3% |
| Q1指引 | +12-17%（高于预期1-2%） |
| 春节酒店 | 双位数增长，价格正增长 |
| 出境游 | 双位数增长，长途游（欧洲澳洲）质量高 |
| International | +60% |

**风险**: 监管问题未明朗，股价脱离基本面

#### 6. 电商

- 唯品会Q1指引0-5%增长（好于Q4的个位数下降）
- 春节晚→1月服饰需求增强
- 即时零售、以旧换新（AR眼镜等智能设备）表现亮眼

#### 7. 体育用品

- Q1销售增长显著高于Q4（春节效应+天气冷）
- 宝胜1月环比+30%（符合季节性）
- **判断**: Q2开始折扣率年对年改善（头部品牌保守定预算，供给下降）

#### 8. 纺织服装（申洲国际、TTI）

**北美市场**:
- 美国服装销售11月+7%，1月+9%（CNBC+NRF数据）
- 库存销售比降至2021年12月低点
- 补库存逻辑存在

**关税影响**:
- 越南印尼关税从20%/19%降至10%，成本压力减轻
- 申洲国际、TTI受益

## 📝 投资结论

1. **消费整体**: 温和恢复，缓慢波动，估值16倍筑底
2. **策略**: 灵活配置，抓住板块轮动
3. **重点板块**:
   - 线下服务: 海底捞、华住/亚朵
   - 价格恢复: 白酒(H2)、化妆品(H2)
   - 供给收缩: 乳制品(Q2后)
   - 海外增长: 申洲国际、TTI

## 风险提示

- 宏观通缩恢复缓慢
- 地产负财富效应拖累
- 监管政策不确定性（OTA）
- 关税政策变化
